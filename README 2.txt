TASA AI Audio to Text & Print Jobs V10 STABLE BACKGROUND

KEY FIXES
- AI transcription runs in a Web Worker, keeping the page responsive.
- Audio is explicitly converted to mono 16 kHz before Whisper receives it.
- The full recording is sent to Whisper with 30-second internal windows and overlap.
- Original transcript, editable copy, email tools and print job fields remain included.

RUN ON COMPUTER
Windows: double-click START_ON_WINDOWS.bat
Mac: right-click START_ON_MAC.command and choose Open
Then use the localhost address shown.

RECOMMENDED
Use Balanced English for normal WhatsApp voice notes.
Use Fast English on slower computers or phones.
First use downloads the selected free model; later use is cached.
