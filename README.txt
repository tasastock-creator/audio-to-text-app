TASA AI AUDIO TO TEXT & PRINT JOBS V12 - SIMPLE BROWSER SETUP

WHAT THIS VERSION DOES
This app runs from a small local web server so it can be opened in any browser on your computer, laptop, or phone on the same Wi-Fi.

BEST WAY TO USE IT
1. Extract this ZIP into a new folder.
2. Run the setup file for your computer once.
3. Start the app with the matching START file.
4. Keep the computer running while you use the app from other devices.

MAC
1. Right-click SETUP_MAC.command and choose Open.
2. Wait until you see: SETUP SUCCESSFUL.
3. Open START_MAC.command.
4. Use http://localhost:8787 on that Mac.

WINDOWS
1. Run SETUP_WINDOWS.bat once.
2. Open START_WINDOWS.bat.
3. Use http://localhost:8787 on that PC.

PHONE
1. Make sure the computer running the app and the phone are on the same Wi-Fi.
2. Open the phone URL shown in the terminal, for example http://192.168.x.x:8787.
3. In Chrome or Safari, use Add to Home Screen / Install App for a quick app-like icon.

If you already ran V11, use FIX_MAC_PYAV.command or simply use SETUP_MAC.command in this fresh V12 folder.

The first transcription model download requires internet. Transcription then runs locally.
